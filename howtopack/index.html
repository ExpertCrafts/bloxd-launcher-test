Creating a Texture Pack

Getting Started

Texture packs allow you to customize the visual appearance of the game by replacing default textures, skies, user interfaces, and 3D models. This comprehensive guide will walk you through the process of creating your own texture pack from scratch.

Setting Up Your Pack Structure

Begin by creating a main folder for your texture pack. Within this folder, you will need to create four essential subdirectories:

Pack Directory Structure

textures

Contains all item and block textures displayed in inventory and gameplay

skyBoxes

Holds six images that create the sky environment surrounding players

css

Stores stylesheet files for customizing user interface elements

models

Includes 3D model files for enhanced in-game objects and structures

Textures Directory

Create the Textures Folder

The textures directory contains all visual elements that appear in the player's inventory and during gameplay. This includes item icons, block surfaces, and various interface elements.

You can create and edit your textures using our dedicated Texture Editor tool, which provides specialized features for game asset creation.

The textures of the game can be found in this repo: Bloxdy/texture-packs

Sky Boxes Directory

Prepare Sky Box Images

The skyBoxes folder requires exactly six photographs to construct a complete sky environment. These images form the faces of a cube that surrounds the player's view:

  • Front view (positive Z-axis)
  • Back view (negative Z-axis)
  • Left view (negative X-axis)
  • Right view (positive X-axis)
  • Top view (positive Y-axis)
  • Bottom view (negative Y-axis)

All six images should maintain consistent dimensions and visual style to create a seamless sky environment.

CSS Customization

User Interface Modification

The css folder enables you to customize the game's user interface and modify the homepage appearance. Through cascading style sheets, you can alter colors, layouts, fonts, and interactive elements.

We recommend using our FaceCG no-code CSS generator, which provides an simple interface for creating custom styles without requiring extensive CSS knowledge.

3D Models Directory

Loading Three-Dimensional Assets

The models directory handles the loading of three-dimensional assets such as enchantment tables, potion stands, decorative objects, and architectural elements. These files define the geometric structure and visual properties of complex in-game objects.

Create and edit your 3D models using our Fac3D editor, specifically designed for game asset creation. This tool provides specialized features for creating models that integrate seamlessly with our platform. You can export the models you make to .GLB so they can be used in bloxd.

Additional Tips and Best Practices

Our Creation Tools

We provide specialized tools to make texture pack creation easier:

  • Fac3D - Our 3D modeling editor for creating game assets
  • Texture Editor - Dedicated tool for creating and editing game textures
  • FaceCG - No-code CSS generator for UI customization
These tools are specifically designed for bloxd and offer features tailored to texture pack creation.

Version Control

Maintain organized backups of your work and keep old versions of your pack, you never know if you need em.

Need Assistance?

If you encounter difficulties during the creation process or have questions about specific implementation details, please reach out for support. Our community and development team are available to assist with technical challenges and creative guidance.

Publishing Your Packs

Once you've created your texture pack, you can share it with the community for others to use and enjoy.

Sharing Your Creation

To publish your texture pack, join our Discord community and contact A_I_X directly. They will help you share your pack with other users and potentially feature it in our workshop.

Also if your pack gets added to the workshop we will advertise it for you!

When reaching out, be prepared to provide:

  • A brief description of your pack's theme and features
  • Screenshots showcasing your work
  • The complete pack files in a compressed format

Creators who contribute quality packs may receive special rewards, Such as ranks or more.
